# AEDS Front Office Enterprise Pack

This module separates operational workflows from Reservations.

## Front Office owns
- Arrival Board
- Departure Board
- In-House Guests
- Room Rack
- Guest Folio
- Cashier
- Night Audit
- Lost & Found
- Guest Messages

## Reservations owns
- Booking Calendar
- Reservation List
- New Reservation
- Availability
- Rate Calendar
- Reservation Reports

## Important
Check-In / Check-Out should be built here, not inside Reservations.
This pack provides the enterprise foundation and safe imports.
